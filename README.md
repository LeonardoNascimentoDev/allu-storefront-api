# Allu Frontstore

Projeto de frente de loja da Allu.

### 🚀 Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
 - Node.js
 - NestJS
 - React.js
 - Next.js
 - MySQL
 - Swagger
 - Docker (Implementando)
 - Jest.js (Implementando)
 - Winton Logger

 ### 📕 Boas práticas e Arquitetura
 - Clean Code
 - Hexagonal Architecture
 - Microservice
 - Unit testing
 - Monitoring and Logs
