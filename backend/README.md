## Framework

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


##  Executando a aplicação com NPM (Caso não queira rodar sem o Docker)

```bash
# instalação
$ npm install

# desenvolvimento
$ npm run start

# watch mode
$ npm run start:dev
```
##  Rodando testes unitários com Jest.js

```bash
# desenvolvimento
$ npm run test

```
