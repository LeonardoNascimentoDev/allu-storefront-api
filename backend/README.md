## Framework

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


##  Executando a aplicação com NPM (caso não queira rodar sem o docker)

```bash
$ npm install

# desenvolvimento
$ npm run start

# watch mode
$ npm run start:dev
```
